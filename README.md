 #ROCK PAPER SCISSORS using REACT JS and CSS

## Setup
1. Install dependencies:
   ```
   npm install
   ```

2. Install Tailwind (already in devDependencies). If Tailwind utilities not available, ensure PostCSS config exists.

3. Run dev server:
   ```
   npm run dev
   ```

## Notes
- Replace the placeholder Lottie JSON at `src/assets/lottie/confetti.json` with a real confetti Lottie file if you want nicer celebration animation.
- Put real sound files at `public/sounds/click.mp3`, `public/sounds/win.mp3`, `public/sounds/lose.mp3`.
- This project uses Vite. You can adapt scripts to Create React App if needed.

Enjoy! 🎉
